py-3rdparty-mediawiki
=====================

.. image:: https://upload.wikimedia.org/wikipedia/commons/thumb/5/54/MediaWiki_logo_1.svg/240px-MediaWiki_logo_1.svg.png
    :alt: MediaWiki logo

This is a wrapper for the `pywikibot <https://pypi.org/project/pywikibot/>`_
E.g. it allows to use `MediaWiki-Japi compatible credential handling <http://mediawiki-japi.bitplan.com>`_
